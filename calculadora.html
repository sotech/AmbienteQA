Calculadora de Impuestos

Informacion

Esta calculadora se encarga de devolver la cantidad de impuestos a pagar segun lo que se vende o compra.

Compra

Al realizar una compra, se cobrara un impuesto de la siguiente manera:

Si se pago con tarjeta un producto con precio menor a $5000, se pagara un 20% de lo pagado

Si se pago con tarjeta un producto con precio mayor a $5000, se pagara un 10% de lo pagado

Si se pago con tarjeta un producto con precio igual a $5000, no habr√° impuesto

Si se pago de contado un producto con precio menor a $3000, se pagara un 5% de lo pagado

Si se pago de contado un producto con precio igual o mayor a $3000, se pagara un 7% de lo pagado

Venta

Al realizar una venta, se debe informar la cantidad de productos vendidos y el precio unitario. Luego se cobra un impuesto a lo vendido:

Si el producto tiene un valor unitario menor o igual a $10000, solo se permite vender hasta 5 unidades. El impuesto sera de un 10% del total

Si el producto tiene un valor unitario mayor a $10000, se permite vender hasta 50 unidades. El impuesto sera de un 50% a lo vendido

Tareas

Verificar que los siguientes casos se cumplan:

  • El pago con tarjeta se calcula correctamente
  • El pago de contado se calcula correctamente
  • La venta se calcula correctamente
  • La venta de productos permite valores dentro de los rangos permitidos
  • Solo se puede introducir numeros a los campos
